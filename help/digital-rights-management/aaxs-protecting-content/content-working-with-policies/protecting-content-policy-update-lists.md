# Listas de atualização de políticas{#policy-update-lists}

Se você atualizar as regras de uso em uma política depois de empacotar o conteúdo, o servidor de licenças precisará ter a versão mais recente para emitir licenças usando a política atualizada. Uma maneira de conseguir isso é através de uma lista de atualização de política. Uma lista de atualização de política é um arquivo que contém uma lista de políticas atualizadas ou revogadas. Quando uma política for atualizada, gere uma nova Lista de atualizações de política e envie periodicamente a lista para fora para todos os servidores de licença.

Para obter mais informações sobre como criar e atualizar políticas, consulte * [Trabalhar com políticas](../../aaxs-protecting-content/content-working-with-policies/content-working-with-policies-overview.md)*.
